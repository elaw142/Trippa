# WELCOME!


# Frontend
frontend is a react-native-expo application.
### Setup
```bash
cd datadons-app
npm install
```
### Running
```bash
npm run {ios or android}
```

# Backend
- install dotnet
```bash
cd datadons-backend
dotnet run
```
find the swagger documentation at localhost:5000/swagger
- you can then utilize the endpoints
